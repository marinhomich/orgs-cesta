# Orgs Cesta
> This project implements the orgs e-commerce basket detail screen. In this screen, static data of the name of the basket, farm, price and basket items are shown.

## Technologies Used
- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
